The Problem

You're having amazing conversations with Claude, but the context gets lost between sessions.

  • Can't find that brilliant idea from last month
  • Recreate the same solutions twice
  • Vendor lock-in with cloud storage
  • Privacy concerns with centralized services

The Solution

haiku-index gives you a complete, searchable archive of every Claude conversation.

  • Index conversations from day one
  • Full-text search in milliseconds
  • 100% local, completely private
  • Zero vendor lock-in, MIT licensed

Get Started in 3 Steps

1. Install Node.js (if you don't have it)

Download from nodejs.org

2. Clone and Install

git clone https://github.com/yourusername/haiku-index.git
cd haiku-index
npm install

3. Export and Index Your Conversations

Export your Claude conversations from Settings ‚Üí Privacy ‚Üí Export Conversations, then:

node src/ingest-all-q-series.js

Then search:

node src/cli.js search "your-query"
